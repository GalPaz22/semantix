# השוואת זרימה: מצב רגיל vs מצב Incremental

## 🔴 מצב רגיל (Full Reprocess)

```
┌─────────────────────────────────────────────────────────┐
│ משתמש לוחץ "Start Reprocessing"                         │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│ handleProcessProducts()                                 │
│ ─────────────────────────────────────────────────────   │
│ incrementalMode: false                                  │
│ reprocessHardCategories: true                           │
│ reprocessSoftCategories: true                           │
│ reprocessTypes: true                                    │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│ reprocess-products.js                                   │
│ ─────────────────────────────────────────────────────   │
│ if (incrementalMode) {                                  │
│   // ❌ לא רץ                                           │
│ }                                                       │
│ else if (options.reprocessSoftCategories) {             │
│   // ✅ רץ!                                             │
│   classify(                                             │
│     text,                                               │
│     name,                                               │
│     categories,        // 🔴 50 קטגוריות               │
│     types,             // 🔴 30 טיפוסים                │
│     softCategories,    // 🔴 300 פילטרים!              │
│     variants,                                           │
│     image                                               │
│   )                                                     │
│ }                                                       │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│ AI Model (Gemini)                                       │
│ ─────────────────────────────────────────────────────   │
│ Prompt:                                                 │
│ "Classify this product:                                 │
│  - Categories: [קטגוריה1, ..., קטגוריה50]              │
│  - Types: [טיפוס1, ..., טיפוס30]                       │
│  - Soft Categories: [פילטר1, ..., פילטר300]"           │
│                                                         │
│ Token Count: ~5,000 tokens 🔴                           │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│ Response:                                               │
│ {                                                       │
│   category: ["חדש"],        // 🔴 מחליף הכל            │
│   type: ["חדש"],            // 🔴 מחליף הכל            │
│   softCategory: ["חדש"]     // 🔴 מחליף הכל            │
│ }                                                       │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│ Database Update:                                        │
│ {                                                       │
│   category: ["חדש"],                                    │
│   type: ["חדש"],                                        │
│   softCategory: ["חדש"],                                │
│   embedding: [new],                                     │
│   description1: "new"                                   │
│ }                                                       │
│                                                         │
│ 🔴 הכל השתנה!                                          │
└─────────────────────────────────────────────────────────┘
```

**עלות:** גבוהה (5,000 tokens × 1,000 products = 5M tokens)  
**זמן:** איטי (30-60 דקות)  
**סיכון:** גבוה (עלול לשנות סיווגים קיימים)

## 🟢 מצב Incremental

```
┌─────────────────────────────────────────────────────────┐
│ משתמש לוחץ "הוסף קטגוריות חדשות (Incremental)"         │
│ עם: ["כחול", "ירוק", "צהוב"]                           │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│ handleIncrementalProcess()                              │
│ ─────────────────────────────────────────────────────   │
│ incrementalMode: true                                   │
│ incrementalSoftCategories: ["כחול", "ירוק", "צהוב"]     │
│ reprocessHardCategories: false    // ✅ מבוטל!          │
│ reprocessSoftCategories: false    // ✅ מבוטל!          │
│ reprocessTypes: false             // ✅ מבוטל!          │
│ reprocessEmbeddings: false        // ✅ מבוטל!          │
│ reprocessDescriptions: false      // ✅ מבוטל!          │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│ reprocess-products.js                                   │
│ ─────────────────────────────────────────────────────   │
│ if (incrementalMode) {                                  │
│   // ✅ רץ!                                             │
│   classifyIncremental(                                  │
│     text,                                               │
│     name,                                               │
│     ["כחול", "ירוק", "צהוב"],  // ✅ רק 3!             │
│     ["אדום", "קיץ"],           // ✅ קיימות             │
│     variants,                                           │
│     image                                               │
│   )                                                     │
│ }                                                       │
│ else if (options.reprocessSoftCategories) {             │
│   // ❌ לא רץ!                                          │
│ }                                                       │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│ AI Model (Gemini)                                       │
│ ─────────────────────────────────────────────────────   │
│ Prompt:                                                 │
│ "Your task: identify which NEW categories apply:        │
│  - NEW categories: [כחול, ירוק, צהוב]  // ✅ רק 3!     │
│  - Product has: [אדום, קיץ] - ADD to these             │
│  Return ONLY from NEW list"                             │
│                                                         │
│ Token Count: ~500 tokens ✅                             │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│ Response:                                               │
│ {                                                       │
│   softCategory: ["כחול"]  // ✅ רק מה שמתאים מהחדשות    │
│ }                                                       │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│ Merge Logic:                                            │
│ existing: ["אדום", "קיץ"]                               │
│ new: ["כחול"]                                           │
│ merged: ["אדום", "קיץ", "כחול"]  // ✅ מיזוג!          │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│ Database Update:                                        │
│ {                                                       │
│   softCategory: ["אדום", "קיץ", "כחול"]  // ✅ רק זה!  │
│ }                                                       │
│                                                         │
│ category: לא השתנה ✅                                    │
│ type: לא השתנה ✅                                        │
│ embedding: לא השתנה ✅                                   │
│ description1: לא השתנה ✅                                │
└─────────────────────────────────────────────────────────┘
```

**עלות:** נמוכה (500 tokens × 1,000 products = 500K tokens = **חיסכון של 90%!**)  
**זמן:** מהיר (5-10 דקות = **חיסכון של 80%!**)  
**סיכון:** אפס (לא משנה שום דבר קיים)

## 📊 השוואה ישירה

| פרמטר | מצב רגיל | מצב Incremental |
|-------|----------|-----------------|
| **פילטרים נשלחים למודל** | 300+ | 2-5 |
| **Tokens לכל מוצר** | ~5,000 | ~500 |
| **עלות ל-1000 מוצרים** | $5-10 | $0.50-1 |
| **זמן ל-1000 מוצרים** | 30-60 דק' | 5-10 דק' |
| **מעדכן category** | ✅ | ❌ |
| **מעדכן type** | ✅ | ❌ |
| **מעדכן softCategory** | ✅ (מחליף) | ✅ (מוסיף) |
| **מעדכן embedding** | אופציונלי | ❌ |
| **מעדכן description** | אופציונלי | ❌ |
| **סיכון לשינוי קיים** | גבוה | אפס |

## 🎯 דוגמה מעשית - 1000 מוצרים

### תרחיש: יש לך 1000 מוצרים מסווגים, רוצה להוסיף 3 צבעים חדשים

#### מצב רגיל:
```
1. בוחר "Soft Categories"
2. לוחץ "Start Reprocessing"

תהליך:
├─ מוצר 1: שולח 300 פילטרים → מקבל סיווג מלא → מחליף הכל
├─ מוצר 2: שולח 300 פילטרים → מקבל סיווג מלא → מחליף הכל
├─ ...
└─ מוצר 1000: שולח 300 פילטרים → מקבל סיווג מלא → מחליף הכל

תוצאה:
✅ כל המוצרים מסווגים מחדש
❌ עלות: $8
❌ זמן: 45 דקות
⚠️ סיכון: 10% מהמוצרים שינו סיווג (לא תמיד לטובה)
```

#### מצב Incremental:
```
1. סומן "מצב הוספה מצטברת"
2. מזין "תכלת, בורדו, שמנת"
3. לוחץ "הוסף קטגוריות חדשות"

תהליך:
├─ מוצר 1: שולח 3 צבעים → מקבל התאמה → מוסיף לקיים
├─ מוצר 2: שולח 3 צבעים → מקבל התאמה → מוסיף לקיים
├─ ...
└─ מוצר 1000: שולח 3 צבעים → מקבל התאמה → מוסיף לקיים

תוצאה:
✅ כל המוצרים קיבלו את הצבעים החדשים (אם מתאימים)
✅ עלות: $0.80
✅ זמן: 7 דקות
✅ סיכון: 0% - שום דבר לא השתנה מלבד הוספת הצבעים החדשים
```

**חיסכון:** 90% בעלות, 84% בזמן, 100% בסיכון!
